# 🏀 Jordan Awwwards Website

Experiência imersiva e interativa apresentando os tênis mais icônicos da linha Air Jordan.

Inspirado em sites premiados do Awwwards, o projeto foca em:

- animações cinematográficas
- design moderno
- experiência premium
- storytelling visual

---

##  Live Website

👉 [https://Jordan - Awwwards.vercel.app](https://jordan-awwwards.vercel.app/)

---

##  Conceito

O projeto mostra a história dos tênis Air Jordan de forma interativa, utilizando animações suaves, vídeos e transições modernas para criar uma experiência luxuosa.

Inspirado na cultura do basquete e no legado de Michael Jordan.

---

## 🚀 Tecnologias

- HTML5
- CSS3
- JavaScript
- GSAP
- ScrollTrigger
